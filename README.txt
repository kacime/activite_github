Ce projet est juste un test pour mettre en pratique l'utilisation de GIT
et de GITHUB. Je réalise une page HTML avec les différentes étapes afin
d'utiliser au mieux les commits.

3 fichiers présents:
- index.html : comme son nom l'indique, on y trouve le code html et l'index du site avec entete, menu, articles et footer
- style.css : règles css pour la mise en page du site
- dossier img : n'abrite pour l'instant que le logo de git pour égayer la page, c'était l'occasion de tester l'ajout de répertoires sur GITHUB
